Notarinos

This is a JavaScript / HTML5 / canvas app to help guitarists learn the fretboard. It uses the Web Audio API library's analyser node to handle the Fast Fourier Transform and most of the additional processing.
I used Harmonic Product Spectrum to further process the data from the FFT for better pitch detection.
The pitch detection has some octave errors (e.g. it might detect A4 instead of A2). The octave errors don't affect the main goal of the app, because it means to show all octaves of a note regardless of the actual octave played, but there are also some non-octave harmonic errors. I'm working on tests to look closer at it.

Instructions

If you play a note, the app shows you all string positions of that note for one octave of each string.
You can change the tuning of any string with the "tuning circles" -- click the top half of the circle to tune up one note, click the bottom half of the circle to tune down one note -- or reset to .
Click the pick to start or stop the app.
